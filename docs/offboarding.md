# Offboarding runbook

When an employee leaves, the offboarding ticket (Jira OPS project) tracks:

1. **Access revocation (within 1-business-day SLA)**
   - Google Workspace account suspended
   - GitHub org membership removed
   - AWS IAM keys revoked
2. **Equipment return** — laptop + hardware key logged on the ticket
3. **Manager sign-off** — approving manager confirms completion

Manifest compiles these into a single auditor-ready evidence package.
